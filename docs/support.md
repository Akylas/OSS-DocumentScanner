## 📞 Get Help

### Issues & Questions

If you need help or found a bug:
1. Check existing [issues](https://github.com/Akylas/OSS-DocumentScanner/issues)
2. Search the documentation
3. Create a new issue with:
   - Clear description
   - Steps to reproduce
   - Screenshots if applicable
   - App version and device info

### Community
