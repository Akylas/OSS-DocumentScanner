# Premiers Pas avec OSS CardWallet

Bienvenue sur **OSS CardWallet**, une application open-source et respectueuse de la vie privée pour stocker et gérer vos cartes de fidélité, cartes de membre, billets et codes-barres.

## Installation

### Android

Vous pouvez installer OSS CardWallet sur Android via :

- **Google Play Store** : [Télécharger depuis Play Store](https://play.google.com/store/apps/details?id=com.akylas.cardwallet)
- **IzzyOnDroid** : [Télécharger depuis IzzyOnDroid](https://apt.izzysoft.de/packages/com.akylas.cardwallet)
- **Versions GitHub** : [Télécharger l'APK depuis GitHub](https://github.com/Akylas/OSS-DocumentScanner/releases)

### iOS

- **App Store** : [Télécharger depuis l'App Store](https://apps.apple.com/app/oss-cardwallet/id6504414362)

## Qu'est-ce que CardWallet ?

OSS CardWallet vous aide à :

- **Stocker les cartes de fidélité** numériquement au lieu de transporter des cartes physiques
- **Scanner les codes-barres** de cartes existantes ou en créer de nouveaux
- **Organiser les cartes** avec des catégories et couleurs personnalisées
- **Accéder rapidement** avec des widgets et actions rapides
- **Sauvegarder en sécurité** sur votre propre stockage cloud

## Démarrage Rapide

### Ajouter Votre Première Carte

1. **Ouvrez l'application** et appuyez sur le bouton **+**
2. **Scannez le code-barres** de votre carte physique
3. **Ajoutez les détails de la carte** comme le nom, les notes et la couleur
4. **Enregistrez** la carte dans votre portefeuille

### Formats de Codes-Barres Supportés

CardWallet supporte de nombreux formats de codes-barres :

- **Codes-Barres 1D** : CODE_128, CODE_39, EAN_13, EAN_8, UPC_A, UPC_E, ITF, CODABAR
- **Codes-Barres 2D** : QR_CODE, DATA_MATRIX, AZTEC, PDF_417

## Fonctionnalités Clés

| Fonctionnalité | Description |
|----------------|-------------|
| Scan de Codes-Barres | Scanner des codes-barres avec votre appareil photo |
| Saisie Manuelle | Entrer les numéros de codes-barres manuellement |
| Organisation des Cartes | Grouper les cartes par catégorie |
| Accès Rapide | Widgets et boost de luminosité |
| Synchronisation Cloud | Synchroniser via WebDAV vers votre serveur |
| Priorité à la Vie Privée | Toutes les données restent sur votre appareil |

## Permissions Requises

Au premier lancement, l'application demandera :

- **Appareil photo** : Requis pour scanner les codes-barres
- **Stockage** (optionnel) : Pour la sauvegarde/restauration
