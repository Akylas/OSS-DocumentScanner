# Premiers Pas avec OSS Document Scanner

Bienvenue sur **OSS Document Scanner**, une application de numérisation de documents open-source et respectueuse de la vie privée pour Android et iOS. Ce guide vous aidera à commencer à numériser, organiser et exporter vos documents.

## Installation

### Android

Vous pouvez installer OSS Document Scanner sur Android via :

- **Google Play Store** : [Télécharger depuis Play Store](https://play.google.com/store/apps/details?id=com.akylas.documentscanner)
- **IzzyOnDroid** : [Télécharger depuis IzzyOnDroid](https://apt.izzysoft.de/packages/com.akylas.documentscanner)
- **Versions GitHub** : [Télécharger l'APK depuis GitHub](https://github.com/Akylas/OSS-DocumentScanner/releases)
- **Obtainium** : Ajouter l'application pour les mises à jour automatiques

### iOS

- **App Store** : [Télécharger depuis l'App Store](https://apps.apple.com/us/app/oss-document-scanner/id6472918564)

## Aperçu de l'Application

![Aperçu de l'application](/img/screenshots/documentscanner/1_en-US.png)

OSS Document Scanner offre un flux de travail optimisé pour :

1. **Capturer** des documents avec l'appareil photo de votre appareil
2. **Éditer** les numérisations avec recadrage, rotation et filtres
3. **Organiser** les documents dans des dossiers
4. **Exporter** en PDF, images ou partager directement
5. **Synchroniser** avec les services cloud (optionnel)

## Démarrage Rapide

### Votre Première Numérisation

1. **Ouvrez l'application** et appuyez sur le bouton appareil photo
2. **Positionnez votre document** dans le viseur
3. **La capture automatique** détectera les bords et prendra la photo, ou appuyez manuellement
4. **Examinez et éditez** l'image capturée
5. **Enregistrez** dans votre bibliothèque de documents

### Permissions Requises

Au premier lancement, l'application demandera les permissions suivantes :

- **Appareil photo** : Requis pour numériser les documents
- **Stockage** : Requis pour enregistrer les documents numérisés
- **Localisation** (optionnel) : Pour géolocaliser les documents
